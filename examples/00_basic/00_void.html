Void script¶

Important

This example shows how to load XDE libraries and how to open an interactive shell at the end of the script. This give the possibility to edit the scene properties (e.g. joints position & velocity) while simulation is running.

source files:

  • main.py

Load XDE libraries¶

At the very beginning of any script, we need to import the XDE loader in order to be able to load XDE libraries. Then we need to load Orocos plugins and typekits specific to XDE. This is done as follows

import loader
import deploy
deploy.loadTypekitsAndPlugins()

Get an interactive shell¶

At any moment of the script, an interactive shell can be launched to access or modify simulation data. This is done as follows

import xdefw.interactive
shell = xdefw.interactive.shell_console()
shell()

print "end of XDE script"

Usually, this method is called at this end of the script, after all agents are running.

Warning

The call of shell() is a blocking procedure, meaning that python interpreter waits for the interactive shell to terminate. The remaining code is executed after that termination, for instance when user calls exit().
